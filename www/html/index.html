Architure

GTSdtracking is an opensource GPS tracking server framework, that enable easy integration of multiple GPS trackers in WEB applications. It provides data acquisition drivers for typical tracker devices or phone's GPS apps. It handle multiple database backend, and support GeoJSON, AIS & NMEA encoding/decoding.

This demo leverages Foundation-5 from Zurb foundation for its template and pages rendering . Leaflet for friendly interactive Maps. Geofields data are served by GpsdTracking GeoJson restfull adapter.
